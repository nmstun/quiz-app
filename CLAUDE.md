# 咲登クイズ(四則演算クイズアプリ)

音声入力で答える算数クイズ。個人開発メモの「クイズアプリ」から着手した最初のプロジェクト。

## 概要

- 四則演算(+ - × ÷)のランダムな問題を出題
- 回答は音声入力(Web Speech API)、非対応環境ではテキスト入力にフォールバック
- 開始時にコース(みじかい5問/ふつう10問/ながい20問)を選択
- 終了後に正解数・所要時間・各問題の結果一覧を表示、結果画面から別のコースを選び直すことも可能
- コースごとのベストスコア(正解数優先、同数なら所要時間が短い方が上位)をlocalStorageに記録
- ビルド不要。`index.html` をブラウザで開くだけで動く静的Webアプリ

## ファイル構成

```
quiz-app/
├── index.html   構造のみ
├── style.css    デザイン
├── script.js    ロジック(問題生成・音声認識・採点)
└── CLAUDE.md    このファイル
```

## 主要ロジック(script.js)

- `generateQuestion()` — 問題生成。減算は常に非負、除算は常に割り切れる組み合わせのみになるよう制約している
- `extractNumber(raw)` — 音声認識/テキスト入力の文字列から数値を抽出
  - 半角/全角の算用数字に対応
  - 簡易的な漢数字パーサ(一〜九、十、百の組み合わせ、0〜999程度)にも対応
  - 「マイナス」を負号として解釈
  - ひらがな(例: 「じゅうさん」)は現状未対応 → 既知の制約
- `submitAnswer()` — 採点し、2秒後に次の問題または結果画面へ遷移
- `TOTAL` — コース選択(`.course-btn`の`data-count`)で決まる出題数。コース選択→`startQuiz()`の流れで書き換わる
- `showResult()` — `localStorage`のキー`sakitoQuizBestScores`にコース(`TOTAL`)ごとのベストスコアを保存。`isBetterScore()`で正解数→所要時間の順に比較

## 動作確認済み事項

- 問題生成を20,000回シミュレーションし、減算の負値・除算の割り切れないケースが0件であることを確認済み
- `extractNumber()` の主要パターン(算用数字/全角数字/漢数字/マイナス表現/文中の数字)をNode.jsでユニットテスト済み(10/10 pass)
